Casos de prueba:

Prueba para cantidad de números pares:
([-1, 4, 2, 3, 10]) --> (res[0], 3) = OK

Prueba para cantidad de números impares:
([-4, 0, 11, 7, 9])   --> (res[1], 3) = OK 
([-7, -5, -40, 8, -3])--> (res[1], 2)=FAILED --> 2!=3

Prueba para cantidad de números positivos:
([-7, -5, 1, 8,-3]) --> (res[2], 2) = OK

Prueba para cantidad de números negativos:
([-7, -5, -40, 8, -3])-->(res[3], 2)=FAILED --> 4!=2

([-7, -5, -40, 8, -3])-->(res[3], 4)=OK
